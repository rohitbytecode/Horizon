## Improvements before putting it on the resume

Current implementation uses **Round Robin** only.

Before starting applying to SDE-2 roles, I'd spend another weekend adding **one or two advanced features**, for example:

- Least Connections load balancing
- Rate limiting
- Config hot reload
- Request retries with exponential backoff
- Circuit breaker
- WebSocket proxying
- HTTPS/TLS termination
- Access logs with structured logging

Adding even one of these will make the project significantly more impressive.
